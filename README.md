# CVRPLIB
Simple repository storing all the CVRP instances from the CVRPLib (accessible at http://vrp.galgos.inf.puc-rio.br/index.php/en/)

It also includes all the (XML100) instances, optimal solutions, as well as the CVRP instances generator from:

Queiroga, E., Sadykov, R., Uchoa, E., & Vidal, T. (2022).
10,000 optimal CVRP solutions for testing machine learning based heuristics.
AAAI’22 Workshop on Machine Learning for Operations Research.
https://openreview.net/pdf?id=yHiMXKN6nTl
